# Results of scanning a subnet of the top 1 million domains (17%) for SRV records
I was testing a script to fetch SRV records (in Microsoft AD this returns Domain Controllers) via DNS. For testing I ran it against a subset of the top 1 million domains. 

The subset was created to use only short domains, since some long subdomains were not expected to result in any useful information. I used `greep -E '^[^.]*\.[^.]*$' domains.txt` to fetch only the ones having at most 1 dot.

All files belonging to the project and its creation have been added.

# Result
The result is rather boring, only 216 returned a record.

# Related work
See https://github.com/realalexandergeorgiev/dns-spray-domain-controllers for a more generic version of the script.
